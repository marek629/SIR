SIR 3.2 Is Out!

First update in 2018 was released!

After two years, first release is out! I'm very happy about that. A round of applause for José M. Ciordia! He provided nice changes in Spanish translation and gave me huge boost up.

Changes comparing to the SIR 3.1 release:

  • Fixed loading transparent image with custom background colour
  • Fixed GIF presence at output file formats list
  • Fixed translations of standard dialog buttons
  • Updated Spanish translation thanks José M. Ciordia
  • Updated Polish translation
